Prawie nienajgorzej:
```java
void CLI() {
    // decay / type erase method
    BST b;
    if ("int") 
        b = new BST<Integer>;
    else if("double")
        b = new BST<Double>;
    else if("string")
        b = new BST<String>;
    else
        throw new Exception("Unsupported type");

    // ale wtedy poniższe polecenie wywali się (ewentualnie, czyli na pewno) dopiero w runtime
    b.insert(5);
}
```
